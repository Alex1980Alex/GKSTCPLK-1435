Документ «Приказ на отпуск продукции (ЗПП-16)»

 

Документ предназначен для формирования приказа на отпуск хлебопродуктов по форме №ЗПП-16. Непосредственно отгрузка оформляется документом «Регистрация транспорта на элеваторе» (вид операции "Отпуск"), который формируется на основании приказа. Предусмотрена возможность автоматического формирования документа «Карточка анализа зерна (ЗПП-47)» на основании приказа на отпуск.

 

Реквизиты шапки документа:

  • Заказ (заявка). Заказ покупателя на отгрузку хлебопродуктов (документ «Заказ покупателя»). Если в настройках параметров элеватора установлено значение флажка "Формирование приказа на отпуск только по заявке", то реквизит "Заказ (заявка)" обязателен для заполнения. В противном случае, заявку на отгрузку указывать не следует. Примечание: заказа покупателя целесообразно использовать в тех случаях, когда требуется отследить оплату (предоплату) клиента за оказанные услуги по отгрузке зерна. В этой ситуации оплата контролируется по заказу. На основании данных заявки заполняются следующие поля приказа: "Организация", "Действует до", "Вид перевозки", "Контрагент", "Договор", "Грузополучатель", "Грузоотправитель", а также табличная часть "Зернопродукты". 
  • Организация. Организация элеватора (справочник «Организация»).
  • Действует до. Дата окончания действия приказа, по истечению которой по данному приказу не возможно оформить отгрузку зернопродуктов.
  • Фонд. Фонд, с которого будет производится отпуск зерна (справочник «Фонды»). Если тип фонда "Собственный", значит приказ оформляется для продажи зерна элеватором контрагенту (продажа через весовую). В данном случае контрагент является покупателем. Если тип фонда "Хранение" - контрагент является владельцем зернопродуктов, который забирает зерно со своего лицевого счета.
  • Вид перевозки. Вид перевозки: автомобильный или железнодорожный транспорт.
  • Контрагент. Юридическое/физическое лицо, с которым заключен договор (поле Заказчик (плательщик) в форме СП-31) (справочник «Контрагенты»).
  • Договор. Договор с контрагентом (справочник «Договоры контрагентов», вид договора "С покупателем"). Если приказ оформляется для продажи зерна контрагенту, то в качестве договора необходимо указать договор продажи. Если приказ составляется для отпуска зерна владельцу, то в качестве договора необходимо указать договор на предоставление услуг, по которому зерно поступало от контрагента на хранение.
  • Грузоотправитель. Грузоотправитель (поле Грузоотправитель в форме СП-31) (справочник «Контрагенты»).
  • Грузополучатель. Грузополучатель (поле Грузополучатель в форме СП-31) (справочник «Контрагенты»).

Закладка «Зернопродукты»

В табличной части указываются культуры для отпуска контрагенту по приказу. Для каждой позиции требуется установить количество отгружаемой культуры и склад, с которого будет производится отпуск. Дополнительно (не обязательно) можно указать показатели качества, с которыми необходимо отгрузить зерно. Состав показателей качества редактируется формой подбора (кнопка "Подбор характеристик"). Первоначальный состав показателей задается в настройках параметров элеватора закладка "Настройка колонок документов". Контроль соответствия фактических показателей качества при отгрузке с установленными в приказе программой не производится.

 

Движения по регистрам. Документ формирует движения по регистру накопления "Отпуск продукции по приказам", по которому контролируется программой количество отпускаемых культур.

Печатные формы. По документу можно распечатать отраслевую форму ЗПП-16.
